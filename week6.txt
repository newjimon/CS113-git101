อาร์เรย์ คือ ชนิดข้อมูลแบบหนึ่งที่นําชนิดข้อมูลพื้นฐาน เช่น
ตัวอักษร(character) เลขจํานวนเต็ม(Integer) เลขทศนิยม
(Floating-point) มาประยุกต์ใช้ให้เป็นชนิดข้อมูลแบบอาร์เรย์

ลักษณะโครงสร้างข้อมูลแบบอาร์เรย์(Array Structure)ต่าง
จากการประกาศตัวแปรทั่วไปในด้านจํานวนการเก็บ ข้อมูลได้
มากว่า ง่ายต่อการเรียกใช้งาน เพราะมีตัวชี้อินเด็กซ์(index)

การประกาศตัวแปรอาร์เรย์ int y[10];
// ตัวแปร y สามารถเรียกใช้งานเพื่อเก็บข้อมูลได้ครั้งละ 10 ข้อมูล

อาร์เรย์ 2 มิติ (Array 2 Dimension)
เป็นอาร์เรย์ของอาร์เรย์1 มิติ
โดยเรียก มิติที่1 ว่า row (เป็นมิติหลัก)และมิติที่2 (อาร์เรย์ของมิติที่1)ว่า column

ตัวอย่างโค้ด

#include <stdio.h>
int main() {
 int n, i, j;
 printf("Enter the number of rows or columns: ");
 scanf("%d",&n);
 int a[n][n];

// Assign value to array a
for(i=1;i<=n;i++){
    a[i-1][0] = i;
}
 // Print all values in array a
 for(i = 0;i<n;i++) {
    for(j=0;j<n;j++)
       printf("%2d ", a[j][0]+i);
    printf("\n");
 }
 return 0;
}
