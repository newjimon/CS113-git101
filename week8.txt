week 8
Link list
Link list จะเก็บ address ของช่องที่ติดกันอยู่ด้วย สามารถมีทั้งแบบ ทางเดียวและแบบไปกลับได้
การสร้าง link list

สร้างกล่องเก็บ หรือ NODE
    กล่องที่เก็บข้อมูลจะเรียกว่า node ซึ่งเราจะใช้ struct สร้างขึ้นมา
    
สร้างฟังก์ชั่น
    สร้างฟังชั้นที่จะนำข้อมูลมาใส่ node และเชื่อม node เข้าด้วยกัน

Function print
การแสดงข้อมูลที่มีอยู่สามารถทำได้โดยเริ่มจากหัวแล้วขยับไปยัง node ต่อไปด้วย pointer ที่เก็บไว้

การเข้าถึงข้อมูลแต่ละตัวใน linked list (Traversal)
โดยมีฟังก์ชั่นที่สร้าง linked list ที่มี 3 โหนด แล้วสร้างอีกฟังก์ชั่นขึ้นมาเพื่อเข้าถึงข้อมูล
โดยการแสดงข้อมูลแต่ละตัวออกมา

Function delete
สามารถลบ node ที่อยู่ภายใน link list ได้โดยเปลียน pointer ให้ข้ามไป node ถัดไปเลย
